# terminal-proxy
Proxy management tool for terminal.

## Install
```plain
pip install terminal-proxy
```

## Usage
### Config
```plain
# If you are on Windows, please run as administrator
proxy config 127.0.0.1:1080
```

### Turn on
```plain
proxy on
```

### Show
```plain
proxy show
```

### Turn off
```plain
proxy off
```
